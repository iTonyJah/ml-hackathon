# Документация по проекту

Этот файл - короткая карта документации.

## Основной маршрут чтения

1. `FINAL_REPORT.md` - итоговый отчет по проекту.
2. `ARCHITECTURE_OVERVIEW.md` - архитектура сервиса.
3. `METRICS_AND_EXPERIMENTS.md` - метрики, эксперименты и результаты.
4. `DEVELOPMENT_STORY.md` - что делали и как пришли к решению.
5. `REPRODUCIBILITY.md` - как воспроизвести запуск.
